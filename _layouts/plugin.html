How to install

  1. Download the .zip file for your platform
  2. Extract the .clap bundle and copy it into your CLAP folder:
    • Windows: C:\Program Files\Common Files\CLAP
    • Mac: /Library/Audio/Plug-Ins/CLAP or ~/Library/Audio/Plug-Ins/CLAP
    • Linux: ~/.clap
    • Or create your own folder and add it in your DAW's plugin settings
  3. Restart your DAW — or use its plugin reload/scan function

Update: Just replace the old .clap file with the new one from the ZIP and reload your DAW.

System Requirements

WindowsWindows 11 64-bit (modern GPU required)
macOSApple Silicon (M1–M4), modern GPU
LinuxModern distro with up-to-date graphics drivers (Mesa/Vulkan)
GPUDirectX 12 / Vulkan / Metal — any GPU from ~2015+
DAWCLAP-compatible (Reaper, Bitwig, Studio One, FL Studio)

⚠ GPU required: LX plugins use GPU-accelerated rendering (Iced + wgpu) for real-time spectrum and meter displays. Old hardware will crash or show blank displays. Windows 10, old Intel Macs, and Linux on older GPUs (no Vulkan support) are not supported. Learn more in the FAQ.
